# Repo bootstrap target

The first coding-agent task should create the initial package and module structure only.

Target packages:

- PaperWMApp
- PaperWMCore
- PaperWMMacAdapters
- PaperWMRuntime

Initial output should include:
- package manifests
- compileable Swift targets
- protocol skeletons
- core data types
- test targets
- placeholder implementations with TODOs

It should not include:
- real AX window control
- event tap implementation
- observer plumbing
- persistence implementation
- advanced UI
